## 11400104 CpuUsage统计异常

**错误信息：**

The status of the system CPU usage is abnormal.

**错误描述：**

当前cpuUsage的状态是异常的。

**可能原因：**

hiview服务进程未正常启动。

**处理步骤：**

检查hiview进程运行状态，重启hiview服务或者重启系统。<!--RP1--><!--RP1End-->
